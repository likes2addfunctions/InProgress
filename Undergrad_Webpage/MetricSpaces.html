Theorems and Skills for Metric Spaces

Bolzano-Weierstrass Theorem

Any bounded sequence in a metric space has a convergent subsequence.

Baire Category Theorem

Suppose $G_{n}$ are countably many open, dense subsets of a metric space $M$, then $\displaystyle \cap_{n} G_{n}$ is still dense in $M$.

A complete metric space can not be written as the complement of such an intersection.

Cantor Set

The traditional Cantor set is made by removing the interval $(\frac{1}{3}, \frac{2}{3})$ from $[0,1]$. Then repeatedly removing the open middle third interval from the leftover pieces.

The Cantor Set it a compact, non-empty, perfect and toally disconnected set.

Given any compact, non-empty metric space $M$, there is a surjection from the Cantor Set onto $M$.

Heine-Borel Theorem

A closed bounded subset of a metric space is compact.

Convergence of Cauchy Sequences

Cauchy Squences converge in metric spaces. A sequence $x_{n}$ is $Cauchy$ if for any $\epsilon >0$ there is $N$ such that $d(x_{n},x_{m}) < \epsilon$ for all $n,m >N$.

Brouwer Fixed point theorem

Let $B^{n}$ denote the closed unit ball in $\mathbb{R}^{n}$, then any continuous function $f: \mathbb{R}^{n} \rightarrow \mathbb{R}^{n}$ has a fixed point.

Contraction Mapping Principle

Let $M$ be a complete metric space and $f:M \rightarrow M$ be a contraction, then $f$ has a unique fixed point $p$ and for any $x \in M$, $$\lim_{n \rightarrow \infty} f^{n}(x) = p$$ where $f^{n}$ denotes the $n$-fold composition $f \circ f \circ \dotd \circ f(x)$
