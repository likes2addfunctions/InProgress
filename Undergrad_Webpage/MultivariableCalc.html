Jensen's Inequality

Suppose that $A \subset \mathbb{R}^{n}$ is convex and that $f$ is convex on $A$. Then for any point $a_{1}, \dots, a_{r} \in A$ and nonnegative constants $\lambda_{1}, \dots, \lambda_{r}$ such that $\displaystyle \sum \lambda_{k} = 1$ one has, $$f(\sum \lambda_{k}a_{k}) \leq \sum \lambda_{k}f(a_{k}$$ $f$ is convex on $A$ if $f(\lambda x + (1-\lambda)y) \leq \lambda f(x) + (1-\lambda)f(y)$ for all $x,y \in A$ and $0 \leq \lambda \leq 1$.

Implicit Function Theorem [Pugh p286]

Suppose $U \subset \mathbb{R}^{n} \times \mathbb{R}^{m}$ and $f: U \rightarrow \mathbb{R}^{m}$ is $C^{r}$. Then for any $(x_{0},y_{0}) \in U$ there is a neighborhood, $V$ of $z_{0} = f((x_{0},y_{0}))$ such that $f$ is the graph of a unique $C^{r}$ function $y = g(x)$ on $V$.

Inverse Function Theorem

Suppose $U \subset \mathbb{R}^{m} \times \mathbb{R}^{m}$ and $f: U \rightarrow \mathbb{R}^{m}$ is $C^{r}$. If at some point $p \in U$, $(Df)_{p}$ is an isomorphism, then there are neighborhoods $U' \ni p$, $V' \ni f(p)$ such that $f$ is a diffeomorphism $U' \rightarrow V'$.

Schwartz Theorem [HK61 p340]

For all $x,y \in \mathbb{R}^{n}$ one has $ |x \cdot y| \leq ||x||\ ||y||$ with equatlity only when there is $a \in \mathbb{R}$ such that $y = ax$

Use of the Hessian

For a $C^{2}$ function $f$ on a convex set $A \subset \mathbb{R}^{n}$ define the Hessian matrix as $\nabla^{2} f = [\partial_{x_{i}}\partial_{x_{j}} f]$. Then $f$ is convex iff $\nabla^{2} f$ is positive definite.

Fundamenetal Theorem for Line Integrals

If $C$ is a smooth curve in $\mathbb{R}^{n}$ given by the vecor equation ${\bf r}(t), a\leq t\leq b$ then $$\int_{C} \nabla f \cdot d {\bf r} = f({\bf r}(b)) - f({\bf r}(a))$$

Green's Theorem

Let $C$ be a positively oriented, sufficiently smooth simple closed curve and let $D$ be the region bounded by $C$. If $P,Q: \mathbb{R}^{2} \rightarrow \mathbb{R}$ have conitnuous partial derivatives on some open set $U \supset D$ then $$\int_{C} P dx + Q dy = \int \int_{D} \left( {\partial Q \over \partial x} - {\partial P \over \partial y} \right) dA $$ More generally, if we write ${\bf F} = P{\bf i} + Q{\bf j}$ and keep the hypotheses above, then $$\int_{C} {\bf F} \cdot {\bf n}\ ds = \int \int_{D} \mbox{div } {\bf F}(x,y)\ dA$$

Gauss's (Divergence Theorem)

Let $E$ be simple solid region with boundary surface $S$. Let ${\bf F}$ is a vector field on $\mathbb{R}^{3}$. If there is some $U \subset \mathbb{R}^{3}$ such that $E \subset U$ and the components of ${\bf F}$ have continuous partial derivatives on $U$, then $$\int \int_{S} {\bf F} \cdot d{\bf S} = \int \int \int _{E} \mbox{div } {\bf F}\ dV $$

Stoke's Theorem

Let $S$ be a sufficiently smooth curve with boundary cuvre $C$. Let ${\bf F}$ is a vector field on $\mathbb{R}^{3}$. If there is some $U \subset \mathbb{R}^{3}$ such that $S \subset U$ and the components of ${\bf F}$ have continuous partial derivatives on $U$, then $$\int_{C} {\bf F} \cdot d{\bf r} = \int \int_{S} \mbox{curl } {\bf F} \cdot d{\bf S}$$

Surface Area of Parameterized surface

Suppose a smooth surface $S$ is parameterized by the equation ${\bf r}(u,v) = (x(u,v), y(u,v), z(u,v))$. Then the surface area of $S$ is given by $$A(S) = \int \int _{D} | {\bf r}_{u} \times {\bf r}_{v} | dA$$
