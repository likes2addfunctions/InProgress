Theorems and Skills from Undergraduate Real Analysis

Cauchy-Schwarz Inequality

$\displaystyle{\left( \sum_{i=1}^{n}a_ib_i \right)^2 \le \left(\sum_{i=1}^{n}a_i^2 \right) \left(\sum_{i=1}^{n}b_i^2 \right)}$

Binomial Theorem

$\displaystyle{(x+y)^n = \sum_{k=0}^{n} {n \choose k}x^{k}y^{n-k}}$

Bolzano-Weirstrass Theorem

Every bounded sequence in a metric space has a convergent subsequence.

Change of Variable for Integration

$\int f(u(x))u'(x) dx = \int f(u) du$

Intermediate Value Theorem

If $f$ is continuous on $[a,b]$ then for any $y$ between $f(a)$ and $f(b)$ there is some $c \in [a,b]$ such that $f(c) = y$

Taylor's Theorem

If $f^{(n)}(t)$ exists for all $t \in (a,b)$ and $f^{(n-1)}(t)$ is continuous on $[a,b]$ then there for any $x, \alpha \in [a,b]$ there is beta in $[a,b]$ such that $\displaystyle f(x) = f(\alpha) + \frac{f'(\alpha)}{1!}(x-\alpha) + \frac{f''(\alpha)}{2!}(x-\alpha)^ + \dots \frac{f^{(n-1)}(\alpha)}{n!}(x-\alpha)^{n-1} +\frac{f^{(n)}(\beta)}{n!}(x-\alpha)^{n} $

Jordan's Lemma

Let $C_{R}$ denote the circle of radius $R$ centerd at the origin of the complex plane. If $f \rightarrow 0$ on $C_{R}$ as $R \rightarrow \infty$ then
$\displaystyle{\int_{R} f(x) dx = 2\pi i \cdot}$ (Sum of Residues of $f$ in the upper half plane)

Mean Value Theorem

For Derivatives

If $f$ is differntiable on $(a,b)$ and continuous on $[a,b]$, then there is some $c \in (a,b)$ such that $f'(c) = \frac{f(b) - f(a)}{b-a}$

For Integrals

If $f,g$ be continuous on $[a,b]$, then there is some $c \in (a,b)$ such that $\displaystyle{\int_{a}^{b} f(t)g(t) dt = f(c)\int_{a}^{b}g(t)dt}$.

Fixed Point Theorem [Rud87 p.220]

If $X$ is a complete metric space and $\phi$ is a contraction $X \rightarrow X$, then there exists a unique $x \in X$ such that $\phi(x) = x$.

$\phi$ is a contraction if $d(\phi(x), \phi(y)) \leq c\ d(x,y)$ for some $c \in \mathbb{R}$ and all $x,y \in X.$

Triangle Inequality

$|x+y| \leq |x| + |y|$

Taylor Expansions of Familiar Functions

$\displaystyle{\sin x = \sum_{k=0}^{\infty} \frac{(-1)^k x^{2k+1}}{(2k+1)!}}$ $\displaystyle{\cos x = \sum_{k=0}^{\infty} \frac{(-1)^k x^{2k}}{(2k)!}}$
   
$\displaystyle{\frac{1}{1-x} = \sum_{k=0}^{\infty} x^{k}}$ $\displaystyle{\ln(1+x) = \sum_{k=1}^{\infty} \frac{(-1)^{k+1} x^{k}}{k}}$

Tests for convergence

Ratio Test

$\displaystyle \sum a_{n}$ converges if $\displaystyle{\limsup_{n \rightarrow \infty} \left| \frac{a_{n+1}} {a_{n}} \right| < 1}$ and diverges if there exists $n_{0}$ such that the ratio is $\geq 1$ for all $n > n_{0}$.

Integral Test

Assume that $f(x) \geq 0 $ and $f$ decreases monotonically on $[k,\infty]$ then $\displaystyle{ \int_{k}^{\infty} f(x) dx}$ converges iff $\displaystyle \sum_{n=k}^{\infty} f(n)$ converges

Root Test

Given $\displaystyle \sum a_{n}$ let $\alpha = \limsup_{n\rightarrow \infty} \sqrt[n]{|a_{n}|}$. Then $a_{n}$ converges if $\alpha < 1$ and diverges if $\alpha > 1$.

Leibniz Criterion

If there exist $b_{n} > 0$ such that $a_{n} = (-1)^{n}b_{n}$ and $|a_{n}|$ decreases monotonically to zero then $\displaystyle \sum a_{n}$ converges.

Weirstrass M-Test

Suppose that $f_{n}$ is a sequence of functions defined on a metric space $E$, the $\sum f_{n}$ converges uniformly on $E$ if there exists $M_{n}$ such that $\sum M_{n}$ converges and $\displaystyle |f_{n}(x)| \leq M_{n}$ for all $x \in E$

L'Hospital's Rule

Suppose $f$ and $g$ are differentiable and $g \neq 0$ on $(a,b)$. Suppose ${f' \over g'} \rightarrow A$ as $x \rightarrow a$. If $f,g \rightarrow 0$ or $g \rightarrow \infty$ as $x \rightarrow a$ then ${f \over g} \rightarrow A$ as $x \rightarrow a$.

Integration by Parts

$\int f'g = fg - \int fg'$

Stone-Weirstrass Approximation

If $f$ is a continuous complex function on $[a,b]$ then there exists a sequence of polynomials $P_{n}$ such that $P_{n} \rightarrow f$ uniformly on $[a,b]$. More generally if $\mathcal{A}$ is any algebra of functions that separates points and vanishes nowhere on a compact set $K$ and continuous function on $K$ can be uniformly approximated by functions of $\mathcal{A}$.

$\mathcal{A}$ vanishes nowhere on $K$ if for any point $x \in K$ there is $f \in \mathcal{A}$ such that $f(x) \neq 0$.

$\mathcal{A}$ separates points on $K$ if for any points $x,y \in K$ there is $f \in \mathcal{A}$ such that $f(x) \neq f(y)$.

Arzela-Ascoli Theorem [Pugh p.214]

Any bounded equicontinuous sequence of functions in $C^{0}([a,b], \mathbb{R})$ has a uniformly convergent subsequence.

Parseval's Identity

Suppose that $f \in L^{2}([-\pi,\pi])$ and can be expressed $f(x) = \sum c_{n} e^{ixn}$. Then $\displaystyle \int_{-\pi}^{\pi} f(x) dx = \sum_{n = -\infty}^{\infty} |c_{n}|^{2}$

Riemann-Lebesgue Lemma [Pugh p.163]

A function $f: [a,b] \rightarrow \mathbb{R}$ is Riemann Integrable iff it is bounded and it's set of discontinuity points is of measure 0.

The Fundamental Theorem of Calculus

Version 1

If $f$ is continuous on $[a,b]$ and $F' = f$ then $\displaystyle{\int _a^b f(x) dx = f(b) - f(a)}$

Version 2

$\displaystyle{f(x) = f(a) + \int_{a}^{x} f'(t) dt}$

Heine Borel

If $M$ is a metric space and $K \subset M$ Then $K$ is compact if and only if $K$ is closed and bounded.
